# 02-Space-Shooter
Project 02 for MSCH-C220, 23 February 2021

## Implementation
Built using Godot 3.2.3
Includes WASD mappings (up, left, down, right), escape to quit, space to shoot Bullet 1, control to shoot Bullet 2.
